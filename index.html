nGantt

nGantt is totally free. Feel free to use it, modify it and distribute it. nGantt is a WPF user control written in C#. Even though it currently is in its early stages it is pretty flexible and has the most common gantt control features. nGantt is based on .Net Framework 4

Here are some features

  • Dynamic and flexible timelines
  • Support for multiple timelines
  • Vertical gridlines (attachable to timeline)
  • 3 types of RowGroups (non headerd, headerd and expandable)
  • Multiple GanttTasks on every GanttRow
  • Different GanttTask selection modes
  • GanttTask progressbar indicator (definable per task)
  • Mouse selectable area for "new GanttTask" type of behavior with owner defined context menu
  • Owner defined context menu for GanttTasks And more...

I'll continue to post updates and here are some things I have plans to add to nGantt

  • Support for milestone indicators
  • "Snap to grid" functionality for selection (with "snap accuracy")
  • Better support for layout customization (useful when hosting in WinForms applications) Next version, will include some of the above mentioned features, will be available at the beginning of October 2012.
